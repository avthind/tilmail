# TOOL BEHAVIOR GUIDE (Stickers, Text, Draw)

Goal: Everything should feel simple, cute, modern, and intuitive across web + mobile.

## üé® 1. STICKERS (Add, Move, Resize, Rotate, Delete)

### How to Add
- User taps/clicks a sticker from the sticker bar.
- Sticker appears centered on canvas (envelope or postcard).
- Auto-selected with resize/rotate handles showing.

### Selecting
**Web:**
- Click on sticker ‚Üí selects it.
- Clicking outside deselects.

**Mobile:**
- Tap sticker ‚Üí selects it.
- Tap outside ‚Üí deselects.

### Moving
**Web:**
- Click + drag.

**Mobile:**
- One finger drag.

### Resizing
**Web:**
- Drag corner handle OR
- Shift + drag for proportional resize (optional).

**Mobile:**
- Two-finger pinch to scale.

### Rotation
**Web:**
- Use rotation handle (a curved arrow on top-right).

**Mobile:**
- Two-finger rotate gesture.

### Deleting
- A small trash icon appears when the sticker is selected.
- Single tap/click deletes.

## ‚úèÔ∏è 2. TEXT TOOL (Adding, Editing, Styling, Movement)

### Adding Text
- Tap "Text" tool ‚Üí a default text box appears ("Your text‚Ä¶").
- Auto-selected with handles.

### Editing Text
**Web:**
- Double click ‚Üí starts editing.

**Mobile:**
- Double tap ‚Üí brings up keyboard.
- Auto-focus the text field.

### Styling Options (minimal)
- Font size: 3 sizes: small / medium / large 
- 4 pastel colors: black, yellow, pink, blue 
- Font weight (regular, bold, italics, underline) 

### Moving / Resizing / Rotation
Same gestures as stickers:
- Move: drag (1 finger)
- Resize: corner handles (web) / pinch (mobile)
- Rotate: rotate handle (web) / pinch rotate (mobile)

### Deleting
- Trash icon when selected.

## üé® 3. DRAW TOOL (Pen/Crayon)

### Drawing Behavior
- Freehand drawing on canvas.
- Works on envelope and postcard.

**Web:**
- Mouse click + drag to draw.

**Mobile:**
- One finger = draw
- Two fingers = move/zoom canvas (do not draw)

### Thickness + Color Options
Minimal:
- 3 sizes: thin / medium / thick
- 4 pastel colors: black, yellow, pink, blue

### Undo/Redo
- Very important for mobile.
- Buttons fixed in toolbar.

### Erasing
- Undo, Redo, Trash (delete) options only

## üì± 5. PLATFORM DIFFERENCES (Web vs Mobile)

### Web Advantages
- Precise cursor ‚Üí easier manipulation.
- Visible handles for resize + rotate.
- Keyboard for typing text.

### Mobile Advantages
- Natural pinch/rotate gestures.
- Touch-friendly bigger UI icons.
- Simpler toolbars.
